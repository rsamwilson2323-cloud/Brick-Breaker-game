# 🧱 Block Breaker Game

A classic arcade-style Block Breaker game built using HTML, CSS, and JavaScript with Canvas API. Control the paddle to bounce the ball, break all bricks, and score points while managing limited lives.

---

## 🚀 Features

- 🧱 Brick breaking gameplay  
- 🎮 Paddle control using keyboard & mouse  
- ⚡ Realistic ball physics and collision detection  
- 📊 Score tracking system  
- ❤️ Lives system with game over condition  
- 🏆 Win condition when all bricks are destroyed  
- 🎨 Clean canvas-based UI  
- ⚡ Smooth animation using requestAnimationFrame  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript (Canvas API)

## 🎯 Controls

- ⬅️ Arrow Left → Move paddle left  
- ➡️ Arrow Right → Move paddle right  
- 🖱️ Mouse → Move paddle smoothly  

---

## 🎯 Gameplay

- Bounce the ball using the paddle  
- Break all bricks to win  
- Each brick increases score  
- Avoid missing the ball  
- Lose all lives = Game Over
